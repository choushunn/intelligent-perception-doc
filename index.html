    使用指北 💡必修课 🧭开发工具 🪀

    # 安装

    # 克隆项目
git clone https://github.com/choushunn/intelligent-perception-doc

# 进入项目所在目录
cd intelligent-perception-doc

# 安装pnpm包管理器
npm i pnpm -g

# 设置pnpm的镜像源为淘宝镜像
pnpm config set registry http://registry.npm.taobao.org

# 安装项目所需的依赖项
pnpm install

    # 使用

    # 运行项目
pnpm run docs:dev
